# 💰 Reajuste Salarial em Python

Este é um script simples feito em **Python** que calcula o novo salário de um trabalhador com base em faixas de rendimento predefinidas. O objetivo do projeto é praticar estruturas condicionais e entrada de dados via terminal.

## 🧠 Lógica do reajuste

O usuário informa seu salário atual e o sistema calcula o novo valor com base nas seguintes regras:

| Faixa Salarial (R$)        | Aumento (%) |
|---------------------------|-------------|
| Até 1.563,00              | 25%         |
| De 1.563,01 até 2.200,00  | 10%         |
| De 2.200,01 até 3.500,00  | 6%          |
| Acima de 3.500,00         | 2%          |
